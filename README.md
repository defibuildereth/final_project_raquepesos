# Raquepesos

Brief - Make an ERC20 token

Make an ERC20 token on the Ethereum Rinkeby testnet. The user should be able to:

MVP

- Connect their metamask wallet to the front end
- Claim some amount of the token per some unit time

Extensions

- 'Buy'/'sell' the token on uniswap
- Stake their tokens on the frontend, either single sided or the token/ETH LP token from uniswap
- View the yearly % return from staking
